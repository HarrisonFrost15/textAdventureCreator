Commands List


North, East, South and West Commands:

'North', 'East', 'South' and 'West' - These commands move you around the map

Hint Command:

'Hint' - This command gives you a hint for when you are stuck.

Break Command:

'Break' - This command allows you to break items. E.g Break Chair

Jump Command:

'Jump' - This command allows you to jump around the map in any direction. E.g Jump West

Attack Command:

'Attack' - This command allows you to attack items with items in your inventory. E.g Attack Window Axe

Drink Command:

'Drink' - This command increases your stamina.

Eat Command:

'Eat' - This command increases your health.

Examine Command:

'Examine' - This command will provide you with a description of an item and tell you the condition that its in.

Open and Close commands:

'Open' + 'Close' - This command will allow you open and close items and reveal any items. E.g open drawer

Take and Drop Command:

'Take' + 'Drop' - This command allows you to put items in and out of your inventory.

Unlock and Lock Command:

'Unlock' + 'Lock' - This command will lock and Unlock different items. It will also lock door to a certain direction E.g. north, east, west and south.

Climb Command:

'Climb' - This command allows you to jump to different directions.

Dig Command:

'Dig' - This command allows you to dig into the ground to reveal and collect a hidden item.

Push Command:

'Push' - This command will allow you to push an item and reveal another item that is hidden.

Fix Command:

'Fix' - This command allows you to fix a broken item.
