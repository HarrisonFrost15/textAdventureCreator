Commands List


North, East, South and West Commands:

'North', 'East', 'South' and 'West' - These commands move you around the map

Hint Command:

'Hint' - This command gives you a hint for when you are stuck.

Break Command:

'Break' - This command allows you to break items. E.g Break Chair

Jump Command:

'Jump' - This command allows you to jump around the map in any direction. E.g Jump West

Attack Command:

'Attack' - This command allows you to attack items. E.g Attack Window

Drink Command:

'Drink' - This command increases your stamina.

Eat Command:

'Eat' - This command increases your health.

Examine Command:

'Examine' - This command will provide you with a description of an item.

Open and Close commands:

'Open' + 'Close' - This command will allow you open and close items E.g open drawer

Take and Drop Command:

'Take' + 'Drop' - This command allows you to put items in and out of your inventory.

Unlock and Lock Command:

'Unlock' + 'Lock' - This command will lock and lock different items. It will also lock door to a certain direction e.g. north, east, west and south.

Climb Command:

'Climb' - This command allows you to jump to different directions.

Dig Command:

'Dig' - This command allows you to into ground to collect an item.
